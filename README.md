# NovaDesign - Mega App per Professionisti dell'Edilizia

Una completa applicazione web per professionisti dell'edilizia di tutte le categorie.

## Funzionalità

- **Gestione Progetti**: Organizza e traccia i tuoi progetti di costruzione
- **Calcolatore Materiali**: Calcola quantità e costi dei materiali
- **Preventivi**: Crea preventivi professionali
- **Pianificazione**: Programma attività e scadenze
- **Gestione Documenti**: Archivia e organizza documenti del progetto
- **Contatti**: Gestisci clienti, fornitori e collaboratori

## Categorie Professionali Supportate

- Architetti
- Ingegneri
- Imprese di costruzione
- Elettricisti
- Idraulici
- Carpentieri
- Muratori
- Piastrellisti

## Come Usare

1. Apri `index.html` nel tuo browser
2. Seleziona la tua categoria professionale
3. Inizia a gestire i tuoi progetti!

## Tecnologie

- HTML5
- CSS3
- JavaScript ES6+
- Responsive Design
